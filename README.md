直接打开 `index.html` 即可在浏览器中游玩，零依赖，纯原生 HTML5 Canvas + JavaScript。

## 📜 游戏简介

《溯世行》是一款以**中华五千年历史**为背景的横版跑酷游戏。

你踏入时空长河，从原始社会的篝火旁出发，穿越青铜时代、百家争鸣、秦汉一统、三分天下、盛世大唐、水墨两宋，直至星辰大海的未来。每当你试图改变历史，都将付出不可逆的代价。

## 🎯 核心玩法

| 操作 | 按键 | 说明 |
|------|------|------|
| 跳跃(二段) | `空格` / `↑` / 点击 | 空中可二段跳 |
| 滑铲 | `Shift` / `S` / `↓` | 躲避高处障碍 |
| 寸光回溯 | `E` | 重置所有障碍，付出记忆代价 |
| 丹心护体 | `Q` | 历代英灵挡伤害(最多3次) |
| 山河移步 | `R` | 清除障碍，但别处出现更多 |
| 妄念改写 | `T` | 减速+短暂无敌，代价痛苦 |
| 天命解封 | `Y` | 清屏无敌，下关双倍伤害 |

## 🧩 特色系统

- **天气影响玩法**: 雨天地滑加速、雪天减速、沙尘暴降低能见度
- **五大改命技能**: 每使用一次改命技能，屏幕边缘撕裂加剧，平行时空趋于不稳定
- **时空涟漪**: 改命次数越多，画面越不稳定（撕裂、重影、闪烁）
- **多结局**: 根据改命次数和收集物，触发不同的结局台词
- **平行时空残影**: 屏幕中飘过的"另一个自己"——那是未曾改命的你
- **叙事系统**: 每关最多5段叙事，边走边看历史
- **移动端支持**: 自动识别触屏设备，显示技能按钮栏

## 🖥️ 技术栈

- HTML5 Canvas (1280×720 原生分辨率)
- 纯 JavaScript (ES6+)，零外部依赖
- CSS3 动画 + Flexbox 布局
- 响应式设计 (自适应移动端)
