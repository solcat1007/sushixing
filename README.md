# 溯世行 — 国风横版跑酷游戏

> 以五千年中华文明为骨，以国风水墨为皮，横版跑酷穿梭 18 个历史时代。
> 从炎黄之战到航母出海，跑过整个华夏的春夏秋冬。

## 游戏背景

「溯世行」是一款纯原生 Canvas 国风横版跑酷游戏。玩家操控主人公穿越中华五千年文明史——从远古炎黄争战、秦汉一统、唐宋盛世，到近现代复兴，共 18 个关卡。每个关卡对应一个历史时代，背景、氛围、难度层层递进。

## 核心玩法

### 基础操作
| 按键 | 功能 |
|------|------|
| 方向键上 / W / Space | 跳跃（可二段跳） |
| 方向键下 / S | 下落加速 |
| 方向键左 / A | 向左移动（空中可微调） |
| 方向键右 / D | 向右移动 |
| Q / E | 切换技能 |
| K 或 Ctrl | 释放技能 |
| P | 暂停/继续 |
| M | 静音/取消静音 |

### 五大改命技能
| 技能名称 | 效果 | 冷却 |
|----------|------|------|
| 时间凝滞 | 减速敌人和障碍物 5 秒 | 20 秒 |
| 空间跃迁 | 短距离瞬移穿墙 | 10 秒 |
| 天人合一 | 无敌护盾 + 加速 3 秒 | 25 秒 |
| 因果逆转 | 死而复生，保留当前分数 | 单次 |
| 万象归宗 | 全屏清怪，大招特效 | 30 秒 |

### 关卡与难度递进
- **1-3 关**（远古-夏商周）：基础教学，二段跳入门
- **4-6 关**（秦汉）：障碍密集化，移动平台引入
- **7-9 关**（三国-隋唐）：敌人追击，复合障碍
- **10-12 关**（宋元）：限时通关，动态地形
- **13-15 关**（明清）：恶劣天气（雨/雪/雾）叠加
- **16-18 关**（近现代）：最终试炼，全要素合集

### 特色系统
- **天气系统**：雨、雪、雾、雷电动态切换，影响移动和视野
- **平行时空残影**：随机触发时空裂隙，生成全屏镜像障碍
- **多结局叙事**：通关后根据技能使用频次和死亡次数触发不同结局
- **Boss 战**：每 3 关一次关底 Boss，需技能组合应对

## 技术特性
- **纯原生 Canvas 渲染**：零外部依赖，不依赖任何游戏引擎
- **自定义物理引擎**：重力/碰撞检测/惯性系统完全手写
- **粒子特效系统**：技能特效、天气粒子、残影效果
- **音频系统**：Web Audio API 动态音效 + 背景音乐
- **移动端适配**：虚拟方向键 + 技能按钮，触屏可玩
- **localStorage 存档**：进度/设置/成就自动保存

## 技术栈
- **渲染**：HTML5 Canvas 2D API
- **物理**：自定义刚体碰撞引擎
- **音效**：Web Audio API + AudioContext
- **存储**：localStorage 持久化
- **动画**：requestAnimationFrame 帧循环
- **构建**：单 HTML 文件，零构建步骤
